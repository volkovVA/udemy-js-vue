# aviatickets-app

### Airline search app

***Goal***: Practicing the layout in the **Materialize** framework. Studying HTTP requests to the server using JavaScript library technology **Axios**. Working with **DOM**. Studying methods of working with data objects.

* Native JavaScript
* Materialize
* Axios
