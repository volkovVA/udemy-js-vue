# news-app
## SPA проект

Новостное приложение с возможностью выбора настроек для поиска новостей.

***Цель***: Практика вёрстки на фреймворке **Materialize**. Изучение HTTP-запросов на сервер
по технологии **XMLHttpRequest**. Работа с **DOM**.

```
native JavaScript
materialize
```
